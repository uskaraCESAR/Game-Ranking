Link do screencast:

- Python
- Django
- Selenium

## Como executar o projeto

1. Crie e ative um ambiente virtual.
2. Instale as dependencias:

```bash
pip install -r requirements.txt
```

3. Execute as migracoes:

```bash
python manage.py migrate
```

4. Inicie o servidor:

```bash
python manage.py runserver
```
